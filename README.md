# Solitario 2048

Juego web desarrollado para la asignatura Programación Orientada a la Web (UCAB). Consiste en colocar cartas numeradas en cuatro columnas, fusionándolas para alcanzar el valor 2048. Desarrollado con HTML5, CSS3 y JavaScript puro sin frameworks. Repositorio: https://github.com/Sebfigueira/ACTIVIDAD-1-Solitario-2048-
